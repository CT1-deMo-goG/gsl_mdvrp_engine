# GSL Engine: MDVRP Module 🚀
A proprietary, high-performance heuristic optimization engine specifically designed for the Multi-Depot Vehicle Routing Problem (MDVRP).

**Core Module:** `GSL_MDVRP_MASTER_V28`

This repository serves as a performance portfolio for the MDVRP architecture, demonstrating world-class execution speed, large-scale stress testing, and benchmark verification.

⚠️ **Disclaimer:** Source code is proprietary and not publicly available. This project is open for research collaboration and industrial applications.

## ⚙️ Execution Environment
Unlike traditional Operations Research benchmarks executed on servers, this engine is validated under highly constrained hardware to prove algorithmic efficiency:
* **Platform:** Android (Mobile)
* **Runtime:** Python via Pydroid 3
* **CPU:** Snapdragon Architecture
* **Constraint Handling:** Fully deterministic feasibility validation

## 📊 Benchmark & Performance Portfolio
The `GSL_MDVRP_MASTER_V28` has been rigorously tested under a **Zero-Tuning Policy** (using a single unified script for all problem sizes) across two major categories:

1. **Extreme Scale Stress Test:** Solved a custom 10,000-node, 100-depot MDVRP network in just **8.8 seconds** on a mobile processor.
2. **Cordeau Benchmark:** Achieved 100% deterministic feasibility across all standard academic instances (p-series and pr-series) in sub-second execution times.
